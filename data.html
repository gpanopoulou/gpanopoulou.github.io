Publicly available code

  • FilTER is a Python program for automatically measuring properties of filamentary structures on astrophysical images. You can read more about the algorithm in: G. V. Panopoulou et al. "13CO filaments in the Taurus molecular cloud", MNRAS, 2014. A public version of FilTER can be found on bitbucket.org at this location.

  • cloudcount is a Python program for measuring the number of HI clouds along the line of sight. It is described in Panopoulou & Lenz, "Maps of the number of HI clouds along the line of sight at high galactic latitude", ApJ, 2020 and is available on github at this location.


Publicly available data

  • The maps of the number of HI clouds along the line of sight, as described in G. V. Panopoulou & D. Lenz, "Maps of the number of HI clouds along the line of sight at high galactic latitude", ApJ, 2020 , can be found at this location.

  • The Polaris Flare stellar polarization catalog, as described in G. V. Panopoulou et al., "Optical polarization map of the Polaris Flare with RoboPol", MNRAS, 2015 , is publicly available on the Centre de Données astronomiques de Strasbourg website at this location.

  • The stellar polarization catalog used to demonstrate a tomographic decomposition in the diffuse ISM, as described in G. V. Panopoulou et al., "Demonstration of Magnetic Field Tomography with Starlight Polarization toward a Diffuse Sightline of the ISM", ApJ, 2019 , is available through ApJ at this location.

  • The stellar polarization catalog in the region of highly polarized dust emission, described in G. V. Panopoulou et al., "Extreme starlight polarization in a region with highly polarized dust emission", A&A, 2019 , is available at this location.
