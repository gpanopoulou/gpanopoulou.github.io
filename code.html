Publicly available code

  • FilTER is a Python program for automatically measuring properties of filamentary structures on astrophysical images. You can read more about the algorithm in: G. V. Panopoulou et al. "13CO filaments in the Taurus molecular cloud", MNRAS, 2014. A public version of FilTER can be found on bitbucket.org at this location.

  • cloudcount is a Python program for measuring the number of HI clouds along the line of sight. It is described in Panopoulou & Lenz, "Maps of the number of HI clouds along the line of sight at high galactic latitude", ApJ in press, 2020 and is available on github at this location.
